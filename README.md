# Heart Disease Classification Project

This repository contains a machine learning project focused on predicting heart disease risk using patient health data.

## Project Objectives
- Classify heart disease presence using healthcare metrics
- Apply structured project management techniques
- Evaluate model performance and document results

## Tools & Technologies
- Python, Jupyter Notebook
- Pandas, NumPy, Scikit-learn
- GitHub Projects for task tracking

## Project Timeline
The project is managed over a 10-week period using issue tracking and a project board.
